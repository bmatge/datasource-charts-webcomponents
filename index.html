Charts builder est une suite d'outils pour créer des visualisations de données conformes au Design System de l'État (DSFR). Générez des graphiques accessibles en quelques clics, connectez vos sources de données, et exportez du code prêt à intégrer dans vos sites.

Comment ça marche ?

1

Connectez vos données

API OpenDataSoft, Grist, ou données manuelles

→
2

Configurez le graphique

Choisissez le type, mappez les champs, ajustez les couleurs

→
3

Exportez le code

Copiez le HTML/JS généré dans votre site

Documentation

Découvrez les composants gouv-widgets et les graphiques DSFR Chart disponibles.

Générateur de graphiques

Créez des graphiques en quelques clics : barres, lignes, camemberts... Connectez vos données et personnalisez l'apparence.

Générateur IA (Albert)

Décrivez en langage naturel le graphique que vous souhaitez. L'IA analyse vos données et génère la visualisation.

Playground

Écrivez du code et voyez le résultat en temps réel. Idéal pour apprendre et expérimenter avec les APIs.

Sources

Connectez vos bases Grist, créez des jeux de données manuels, ou liez des APIs pour alimenter vos graphiques.

Vde

Visual Dashboard Editor : éditeur visuel de tableaux de bord.
