# public/

Assets statiques servis tels quels par le serveur de dev Vite et en production.

## Contenu

- `mock-api/` -- Reponses API simulees (fichiers JSON de donnees de test)
- `monitoring-data.json` -- Exemple de format pour les donnees de monitoring

## Notes

Les fichiers de ce dossier sont accessibles directement a la racine du serveur (ex: `/mock-api/data.json`). Ils ne sont pas transformes par Vite lors du build.
