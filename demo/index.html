Bibliotheque de composants

gouv-widgets est une bibliotheque de Web Components pour creer des visualisations de donnees conformes au Design System de l'Etat (DSFR).

Nos composants

Ces composants sont developpes specifiquement pour ce projet et s'integrent avec le systeme de data-bridge.

gouv-source Custom

Connecteur de donnees invisible qui recupere des donnees depuis une API et les distribue aux autres composants.

Voir la documentation

gouv-kpi Custom

Affiche une valeur cle (KPI) avec formatage, couleurs conditionnelles et icones.

Voir la documentation

gouv-datalist Custom

Tableau de donnees avec recherche, filtres, tri, pagination et export CSV.

Voir la documentation

gouv-dsfr-chart Custom DSFR Chart

Wrapper connectant les graphiques DSFR Chart au systeme gouv-source pour des donnees dynamiques.

Voir la documentation

Graphiques DSFR Chart

Ces graphiques proviennent de la bibliotheque officielle DSFR Chart. Ils peuvent etre utilises directement ou via notre wrapper gouv-dsfr-chart.
